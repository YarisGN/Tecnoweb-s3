# Tecnoweb-s3

Proyecto universitario desarrollado con Bootstrap para un diseño responsivo y moderno, integrando componentes clave y lógica de negocio (JavaScript) en diversas secciones funcionales:

![Preview](images/md/img1.jpg)

Inicio: Muestra un Carrusel (Carousel) de presentación interactiva.
![Preview](images/md/img2.jpg)

Quiénes Somos: Incluye una Tabla con módulo de cálculo dinámico para determinar notas definitivas.
![Preview](images/md/img3.jpg)

Servicios: Utiliza un Formulario para simular y ejecutar el cálculo de nómina (pago a trabajador).
![Preview](images/md/img4.jpg)

Contáctanos: Presenta un Mapa Geográfico incrustado para la ubicación.
![Preview](images/md/img5.jpg)

Nuevo: Destaca contenido utilizando el componente Card (Card).
![Preview](images/md/img6.jpg)
